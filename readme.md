# message-create (projeto teste)

Este é um **projeto teste** criado com o objetivo de **estudo e prática** (experimentação) de HTML, CSS e JavaScript.

A ideia é ter uma interface simples para **criar e remover mensagens**, onde a `div#res` funciona como uma **área de postagens** (listagem das mensagens adicionadas).

## Objetivo

- Praticar manipulação do DOM com JavaScript
- Treinar estrutura básica de HTML
- Aplicar estilos com CSS (layout, responsividade e aparência elegante)
- Simular um pequeno “mural” de mensagens (posts)

## Como executar

1. Instale as dependências (se estiver usando Vite):
   ```bash
   npm install
   ```

2. Rode o projeto em modo desenvolvimento:
   ```bash
   npm run dev
   ```

3. Abra no navegador a URL exibida no terminal (ex.: `http://localhost:5173`).

## Estrutura (resumo)

- `index.html`: tela principal com os inputs e a área de postagens
- `src/main.js`: lógica do projeto (adicionar/remover mensagens)
- `style.css`: estilos da interface

## Observações

Este repositório é apenas para **fins educacionais**.  
Sinta-se à vontade para modificar, testar ideias e evoluir a interface/funcionalidades.
